### `agents/examples`
Directory containing example agent configurations in YAML frontmatter format.

---

### `default.md`
General-purpose coding agent with full permissions. Mode: all. Color: #4CAF50.

#### Algorithm
Load agent from `.md` file with YAML frontmatter. Parse frontmatter fields (name, model, tools, permissions, description, mode, color, parent_agent). Extract body after second `---` as system prompt. Use filename stem if name not set. Apply defaults: empty tools list, empty permissions map. Load all `.md` files in directory into agents map keyed by name (last wins).

#### Data
- **Structures:** Agent { Name, SystemPrompt, Model, Tools[], ParentAgent, Permissions{}, ConfigPath, Description, Mode, Color, AgentPerms* }
- **Types:** AgentPermissions { Read{}, Write{}, Edit{}, Bash{}, Glob, Glob, Task{}, Skill{}, WebSearch, WebFetch, CodeSearch, Question }. frontmatter struct with YAML tags.
- **Flow:** File path → read content → parse frontmatter (split at `---`) → YAML unmarshal → create Agent struct → trim body → return map.
- **Permissions:** read: allow all except git/node_modules/.env*, write/edit/bash: ask per change, glob/grep/codesearch: allow, task/websearch/webfetch: allow.

#### Format
Uses YAML frontmatter delimited by `---`. Body becomes system prompt. Loader parses via `internal/agent/loader.go`. Agent struct defined in `internal/agent/types.go`.

### `researcher.md`
Research subagent. Tools: web_fetch, memory_save, memory_recall. Mode: subagent. Color: #2196F3. Parent: default.

#### Algorithm
Same loader algorithm as default.md. Specialized for web-based info gathering. Denies delegate permission.

#### Data
- **Permissions:** web_fetch: allow, memory_save: allow, memory_recall: allow, delegate: deny.
- **Workflow:** Fetch URLs → extract facts → save findings → recall existing data → synthesize response.
- **Constraints:** Only fetch user-provided URLs, mention sources, avoid duplicate fetches.

---

### `analyst.md`
Analytical subagent. Tools: memory_save, memory_recall, delegate. Mode: subagent. Color: #FF9800. Parent: default.

#### Algorithm
Same loader algorithm. Specialized for data reasoning and delegation to other agents.

#### Data
- **Permissions:** memory_save: allow, memory_recall: allow, delegate: allow, web_fetch: deny.
- **Workflow:** Recall stored facts → break problems → delegate to researcher → synthesize structured analysis → save conclusions.
- **Constraints:** State assumptions, present pros/cons, use quantitative reasoning, never fabricate data.

---

### `summarizer.md`
Pure text summarization agent. No tools. Mode: tool. Color: #9C27B0. Parent: default.

#### Algorithm
Same loader algorithm. Minimal agent with empty tools and permissions.

#### Data
- **Permissions:** Empty {}.
- **Workflow:** Read text → identify main topic/key points → produce concise summary (~20% length).
- **Format:** Topic, Key Points (bullets), Conclusion/Takeaway. Never add external info.
