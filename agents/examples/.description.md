### `agents/examples`
Directory containing example agent configurations.

---

### `default.md`
Default agent configuration in opencode frontmatter format. Demonstrates the
structured permission block (`permission:` with `read`/`write`/`edit`/`bash`/
`glob`/`grep`/`task`/etc.), `mode`, `color`, `description`, and a full system
prompt body. This is the fallback agent referenced by `gopengai.json` via
`"default_agent": "default"`.

#### Format
Uses YAML frontmatter delimited by `---` with both native gopengai fields
(`name`, `model`, `tools`, `parent_agent`, `permissions`) and opencode-style
fields (`description`, `mode`, `color`, `permission`). Parsed by
`internal/agent/loader.go`.
