### `agents`
Directory containing agent definitions, system default (`default.md`), and example configurations (`examples/`).

### `default.md`
System default agent config. Referenced by `gopengai.json` as `"default_agent": "default"`. Defines allowed tools: web_fetch, memory_save, memory_recall, delegate. Mode: all. Color: #4CAF50.

#### Algorithm
Load agent from `.md` file with YAML frontmatter. Parse frontmatter fields (name, tools, permissions, description, mode, color). Extract body after second `---` as system prompt. Use filename stem if name not set. Apply defaults: empty tools list, empty permissions map. Load all `.md` files in directory into agents map keyed by name (last wins).

#### Data
- **Structures:** Agent { Name, SystemPrompt, Tools[], Permissions{}, Description, Mode, Color }
- **Types:** AgentPermissions { web_fetch, memory_save, memory_recall, delegate }
- **Flow:** File path → read content → parse frontmatter → YAML unmarshal → create Agent struct → return map.

### `examples/`
Directory containing example agent configurations (OpenCode default, researcher, analyst, summarizer).
