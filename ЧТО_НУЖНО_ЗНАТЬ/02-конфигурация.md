# Конфигурация проекта (internal/config)

## Файл конфигурации

Проект использует JSON-файл `gopengai.json` для настройки. Пример — в `gopengai.json.example`.

### Структура Config

```go
type Config struct {
    Server       ServerConfig `json:"server"`
    LLM          LLMConfig    `json:"llm"`
    AgentsDir    string       `json:"agents_dir"`
    DataDir      string       `json:"data_dir"`
    DefaultAgent string       `json:"default_agent"`
}

type ServerConfig struct {
    Host string `json:"host"` // По умолчанию "0.0.0.0"
    Port int    `json:"port"` // По умолчанию 8080
}

type LLMConfig struct {
    Provider      string `json:"provider"`       // По умолчанию "openai"
    BaseURL       string `json:"base_url"`        // По умолчанию "https://api.openai.com/v1"
    APIKey        string `json:"api_key"`         // Обязательное поле
    Model         string `json:"model"`           // По умолчанию "gpt-4o-mini"
    MaxIterations int    `json:"max_iterations"`  // По умолчанию 10
}
```

### Приоритет настроек (от низшего к высшему)

1. **Значения по умолчанию** (hardcoded в `applyDefaults`)
2. **JSON-файл** `gopengai.json`
3. **Переменные окружения** (env vars, с префиксом `GOPENGAI_`)
4. **Флаги командной строки** (в будущем) — пока не реализованы

### Переменные окружения

| Переменная | Переопределяет | Пример |
|-----------|---------------|--------|
| `GOPENGAI_HOST` | `server.host` | `"127.0.0.1"` |
| `GOPENGAI_PORT` | `server.port` | `"9090"` |
| `GOPENGAI_LLM_API_KEY` | `llm.api_key` | `"sk-..."` |
| `GOPENGAI_LLM_BASE_URL` | `llm.base_url` | `"https://api.anthropic.com/v1"` |
| `GOPENGAI_LLM_MODEL` | `llm.model` | `"claude-sonnet-4"` |
| `GOPENGAI_LLM_PROVIDER` | `llm.provider` | `"anthropic"` |
| `GOPENGAI_LLM_MAX_ITERATIONS` | `llm.max_iterations` | `"20"` |
| `GOPENGAI_AGENTS_DIR` | `agents_dir` | `"./custom_agents"` |
| `GOPENGAI_DATA_DIR` | `data_dir` | `"./data"` |
| `GOPENGAI_DEFAULT_AGENT` | `default_agent` | `"researcher"` |

### Как загружается конфиг

```go
func Load(path string) (*Config, error) {
    // 1. Читаем JSON-файл
    data, err := os.ReadFile(path)
    
    // 2. Парсим в Config
    var cfg Config
    json.Unmarshal(data, &cfg)
    
    // 3. Применяем defaults
    applyDefaults(&cfg)
    
    // 4. Переопределяем env vars
    applyEnvOverrides(&cfg)
    
    return &cfg, nil
}
```

### Пример gopengai.json

```json
{
  "server": {
    "host": "0.0.0.0",
    "port": 8080
  },
  "llm": {
    "provider": "openai",
    "base_url": "https://api.openai.com/v1",
    "api_key": "sk-...",
    "model": "gpt-4o",
    "max_iterations": 10
  },
  "agents_dir": "./agents",
  "data_dir": "./.gopengai",
  "default_agent": "default"
}
```

> **Важно:** `api_key` должен быть установлен либо в JSON, либо через `GOPENGAI_LLM_API_KEY`. Если ключ пустой, LLM-запросы будут падать с 401.
