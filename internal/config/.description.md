### `internal/config`
Directory containing configuration management logic.

---

### `config.go`
Configuration loader — reads `gopengai.json` and applies defaults.

#### Exports
- `Config` struct (`Server`, `LLM`, `AgentsDir`, `DataDir`, `DefaultAgent`)
- `ServerConfig` struct (`Host`, `Port`)
- `LLMConfig` struct (`Provider`, `BaseURL`, `APIKey`, `Model`, `MaxIterations`)
- `Load(path string) (*Config, error)` — JSON file reader with sensible defaults
