### `internal/config`
Directory containing config management logic.

---

### `config.go`
JSON config loader — reads file, applies defaults, env overrides.

#### Algorithm
Read JSON file → unmarshal → apply defaults (zero-values) → apply env overrides → return Config. Priority: JSON < defaults < env vars. Env vars parse int for port/iterations.

#### Data
- **Structs:**
  - `Config` { Server: ServerConfig, LLM: LLMConfig, AgentsDir, DataDir, DefaultAgent }
  - `ServerConfig` { Host, Port }
  - `LLMConfig` { Provider, BaseURL, APIKey, Model, MaxIterations }
- **Types:** String/int fields mapped to JSON keys
- **Flow:** file path → read → json.Unmarshal → applyDefaults → applyEnvOverrides → return *Config

#### Defaults
- Server: Host="0.0.0.0", Port=8080
- LLM: Provider="openai", BaseURL="https://api.openai.com/v1", Model="gpt-4o-mini", MaxIterations=10
- AgentsDir="./agents", DataDir="./.gopengai", DefaultAgent="default"

#### Env Overrides
GOPENGAI_HOST, GOPENGAI_PORT, GOPENGAI_LLM_API_KEY, GOPENGAI_LLM_BASE_URL, GOPENGAI_LLM_MODEL, GOPENGAI_LLM_PROVIDER, GOPENGAI_LLM_MAX_ITERATIONS, GOPENGAI_AGENTS_DIR, GOPENGAI_DATA_DIR, GOPENGAI_DEFAULT_AGENT
