### `internal/config`
Directory containing configuration management logic.

---

### `config.go`
Configuration loader — reads JSON config from file path, applies sensible defaults, and supports environment variable overrides.

#### Exports
- `Config` struct (`Server`, `LLM`, `AgentsDir`, `DataDir`, `DefaultAgent`)
- `ServerConfig` struct (`Host`, `Port`)
- `LLMConfig` struct (`Provider`, `BaseURL`, `APIKey`, `Model`, `MaxIterations`)
- `Load(path string) (*Config, error)` — JSON file reader with env var override support

#### Defaults Applied
- `host`: `0.0.0.0`
- `port`: `8080`
- `provider`: `openai`
- `base_url`: `https://api.openai.com/v1`
- `model`: `gpt-4o-mini`
- `max_iterations`: `10`
- `agents_dir`: `./agents`
- `data_dir`: `./.gopengai`
- `default_agent`: `default`

#### Environment Variable Overrides
- `GOPENGAI_HOST` — server host
- `GOPENGAI_PORT` — server port
- `GOPENGAI_LLM_API_KEY` — LLM API key
- `GOPENGAI_LLM_BASE_URL` — LLM base URL
- `GOPENGAI_LLM_MODEL` — LLM model
- `GOPENGAI_LLM_PROVIDER` — LLM provider
- `GOPENGAI_LLM_MAX_ITERATIONS` — maximum iterations
- `GOPENGAI_AGENTS_DIR` — agents directory path
- `GOPENGAI_DATA_DIR` — data directory path
- `GOPENGAI_DEFAULT_AGENT` — default agent name

#### Priority
`JSON file < defaults < env vars`
