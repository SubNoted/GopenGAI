### `internal/api`
Directory containing API handlers and routing.

---

### `events.go`
SSE event stream implementation stub (future feature).

#### Exports
- None (1 byte - package declaration only)

### `handler.go`
HTTP request handler implementation with session management and chat endpoints.

#### Exports
- `Handler` struct (`LLM *llm.Client`, `DB *db.Queries`, `Config *config.Config`)
- `ChatRequest` struct (`Model`, `Messages []llm.Message`, `Tools []llm.ToolDefinition`, `ToolChoice json.RawMessage`)
- `CreateSessionRequest` struct (`Title`, `AgentName`)
- `ChatResponse` struct (`SessionID`, `MessageID`, `Role`, `Content`, `Model`, `Usage *llm.Usage`, `Error`)
- `SessionView` struct (`ID`, `AgentName`, `Title`, `Status`, `MessageCount`, `CreatedAt`, `UpdatedAt`, `Messages`)
- `MessageView` struct (`ID`, `Role`, `Content`, `AgentName`, `CreatedAt`)
- `(h *Handler) HandleCreateSession(w http.ResponseWriter, r *http.Request)` — POST `/session`
- `(h *Handler) HandleListSessions(w http.ResponseWriter, r *http.Request)` — GET `/session`
- `(h *Handler) HandleGetSession(w http.ResponseWriter, r *http.Request)` — GET `/session/{id}`
- `(h *Handler) HandleDeleteSession(w http.ResponseWriter, r *http.Request)` — DELETE `/session/{id}`
- `(h *Handler) HandleChatMessage(w http.ResponseWriter, r *http.Request)` — POST `/session/{id}/message`
- `(h *Handler) HandleHealth(w http.ResponseWriter, r *http.Request)` — GET `/health`
- `(h *Handler) HandleChatCompletion(w http.ResponseWriter, r *http.Request)` — POST `/v1/chat/completions`

#### External Dependencies
- `./internal/config`
- `./internal/db`
- `./internal/llm`

### `middleware.go`
API middleware implementation stub (future: logging, CORS, recovery, auth).

#### Exports
- None (1 byte - package declaration only)

### `routes.go`
API route registration — wires handler methods to `http.ServeMux` using Go 1.22+ method routing.

#### Exports
- `RegisterRoutes(mux *http.ServeMux, h *Handler)` — registers all endpoints:
  - `GET /health`
  - `POST /session`, `GET /session`, `GET /session/{id}`, `DELETE /session/{id}`
  - `POST /session/{id}/message`
  - `POST /v1/chat/completions`
