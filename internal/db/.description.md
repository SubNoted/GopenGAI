### `internal/db`
Directory containing database connectivity, SQL migrations, sqlc-generated query code, and SQL query definitions.

---

### `connect.go`
SQLite database connection using `ncruces/go-sqlite3` (pure Go, no CGo).

| Function | Description |
|----------|-------------|
| `Open(path string) (*sql.DB, error)` | Opens SQLite at path, applies pragmas (`foreign_keys=ON`, `journal_mode=WAL`, `page_size=4096`, `cache_size=-8000`, `synchronous=NORMAL`, `busy_timeout=5000`), configures pooling (`MaxOpenConns=1`, `MaxIdleConns=1`, `SetConnMaxLifetime(time.Hour)`)

### `embed.go`
Goose migration runner with embedded SQL migrations.

| Function | Description |
|----------|-------------|
| `Migrate(db *sql.DB) error` | Runs all pending Goose migrations from embedded `migrations/` directory |

### `migrations/`
Directory containing SQL migration files for Goose.

#### `001_initial.sql`
Complete SQL migration file (124 lines) creating 5 tables: sessions, messages, agents, memory, delegation_logs. Includes 5 triggers and 4 indexes. Uses Goose migration directives with `-- +goose Up` and `-- +goose Down` sections. The Down section drops tables in reverse FK order (delegation_logs, memory, agents, messages, sessions) to avoid foreign key constraint violations.

### `sql/`
Directory containing raw SQL query definition files for `sqlc` code generation.

#### `agents.sql`
Query definitions for the `agents` table — CreateAgent, GetAgent, ListAgents, DeleteAgent.

#### `delegation_logs.sql`
Query definitions for the `delegation_logs` table — CreateDelegationLog, ListDelegationLogsBySession.

#### `memory.sql`
Query definitions for the `memory` table — CreateMemory, GetMemory, ListMemoryByAgent, DeleteMemory.

#### `messages.sql`
Query definitions for the `messages` table — CreateMessage, GetMessage, ListMessagesBySession, GetBranchFromRootTo (recursive CTE with disambiguated column references), GetAllLeaves, UpdateMessage, DeleteMessage, DeleteSessionMessages.

#### `sessions.sql`
Query definitions for the `sessions` table — CreateSession, GetSessionByID, ListSessions, UpdateSession, DeleteSession.

### Generated files (by `sqlc generate`)
| File | Description |
|------|-------------|
| `db.go` | Queries struct, DBTX interface, WithTx, Prepare, Close |
| `models.go` | Go structs for all 5 tables (Agent, DelegationLog, Memory, Message, Session) with json tags |
| `querier.go` | Querier interface with all CRUD methods |
| `*.sql.go` | Per-query implementations (agents, sessions, messages, memory, delegation_logs) |
