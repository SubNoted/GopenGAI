### `internal/db`
Database connectivity, migrations, sqlc-generated query code.

---

### `connect.go`
Open SQLite connection (pure Go, ncruces driver).

#### Algorithm
Opens SQLite DB at path. Sets pragmas: `foreign_keys=ON`, `journal_mode=WAL`, `page_size=4096`, `cache_size=-8000`, `synchronous=NORMAL`, `busy_timeout=5000`. Sets pool: `MaxOpenConns=1`, `MaxIdleConns=1`, `SetConnMaxLifetime(1h)`.

#### Data
- **Flow:** path → `sql.Open()` → exec pragmas → set pool → return `*sql.DB`

---

### `embed.go`
Migrate DB with embedded Goose migrations.

#### Algorithm
Embed `migrations/*.sql` via `//go:embed`. Set Goose dialect to `sqlite3`. Call `goose.Up(db, "migrations")` to run pending migrations.

#### Data
- **Flow:** `*sql.DB` → set FS/dialect → run Up → return error

---

### `db.go`
sqlc-generated Queries wrapper.

#### Algorithm
New/Prepare: Create `Queries` struct, prepare all statements. WithTx: helper for transactions.

#### Data
- **Structures:** `Queries` (db DBTX + prepared stmts), `DBTX` interface (Exec/Query/Prepare)
- **Flow:** db → New/Prepare → Queries → methods

---

### `models.go`
sqlc-generated structs for tables.

#### Algorithm
Defines Go structs for: `Agent`, `DelegationLog`, `Memory`, `Message`, `Session` with JSON tags.

#### Data
- **Agent fields:** name, system_prompt, tools, model, parent_agent, permissions, config_path, loaded_at
- **DelegationLog fields:** id, parent_message_id, child_agent_name, child_session_id, task_description, result_summary, duration_ms, created_at
- **Memory fields:** id, agent_name, key, value, category, created_at, updated_at
- **Message fields:** id, session_id, parent_id, role, parts, content, agent_name, tool_name, tool_call_id, tool_args, model, token_count, created_at, updated_at, finished_at
- **Session fields:** id, parent_session_id, agent_name, title, active_leaf_id, status, message_count, prompt_tokens, completion_tokens, cost, updated_at, created_at, summary_message_id

### `migrations/001_initial.sql`
Goose migration creating 5 tables + triggers + indexes.

#### Algorithm
Creates tables: sessions, messages, agents, memory, delegation_logs. Includes triggers: updated_at (sessions, messages, memory), message_count (insert, delete). Uses `-- +goose Up/Down`. Down drops in reverse FK order.

#### Data
- **Tables:** sessions, messages, agents, memory, delegation_logs
- **Triggers:** 5 triggers (updated_at x3, message_count x2)

---

### `migrations/002_unique_memory.sql`
Goose migration: adds `UNIQUE(agent_name, key)` index on memory table, removes duplicates first.

#### Algorithm
Deletes duplicate rows keeping latest. Creates `idx_memory_agent_name_key` unique index. Prevents TOCTOU race from concurrent `MemorySave` calls.

#### Data
- **Index:** `idx_memory_agent_name_key` on `memory (agent_name, key)`
