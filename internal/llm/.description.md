### `internal/llm`
Directory containing LLM (Large Language Model) client and streaming logic.

---

### `client.go`
LLM client implementation — HTTP POST to OpenAI-compatible `/chat/completions` endpoint.

#### Exports
- `Client` struct with `BaseURL`, `APIKey`, `Model`, `HTTPClient` fields
- `NewClient(baseURL, apiKey, model string) *Client`
- `(c *Client) ChatCompletion(ctx context.Context, messages []Message) (*ChatCompletionResponse, error)`

### `stream.go`
LLM stream handling stub (future feature — SSE parsing).

#### Exports (optional)
- None (0 bytes)

### `types.go`
LLM type definitions — OpenAI-compatible request/response structs.

#### Exports
- `ChatCompletionRequest` struct (`Model`, `Messages` with JSON tags matching OpenAI API)
- `Message` struct (`Role`, `Content`)
- `ChatCompletionResponse` struct (`ID`, `Choices`, `Usage`, `Error`)
- `Choice` struct (`Index`, `Message`)
- `Usage` struct (`PromptTokens`, `CompletionTokens`, `TotalTokens`)
- `APIError` struct (`Message`, `Type`)
