### `cmd`
Directory containing command-line interface and API entry points.

---

### `api/`
Directory containing the API server entry point. Main: `main.go` loads config, initializes SQLite database with migrations, creates LLM client, wires DB queries and config into the handler, registers routes, and starts HTTP server.

#### `main.go`
API server entrypoint — loads config, initializes SQLite database with migrations, creates LLM client, wires DB queries and config into the handler, registers routes, and starts HTTP server.

#### Exports
- `main()` — starts `GoPengAI listening on <host>:<port>`

---

### `cli/`
Directory containing the CLI entry point. Fully implemented with Cobra commands for chat (single + REPL) and session management (list, show, create, delete).

#### `main.go`
CLI entry point — Cobra framework with root command, chat command (single message or interactive REPL), session management commands (list, show, create, delete), HTTP helpers, and JSON response types.

#### Exports
- `main()` — CLI entry point

#### External Dependencies
- `github.com/spf13/cobra`
