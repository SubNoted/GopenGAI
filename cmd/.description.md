### `cmd`
Directory containing command-line interface and API entry points.

---

### `api/`
Directory containing the API server entry point. MVP: `main.go` loads config, creates LLM client + handler, starts HTTP server.

#### `main.go`
MVP server entrypoint — config loading, LLM client creation, route registration, HTTP server startup.

#### Exports
- `main()` — starts `GoPengAI MVP listening on <host>:<port>`

### `cli/`
Directory containing the CLI entry point (not yet implemented — Phase 8).
