### `cmd/api`

Directory with API server entry point.

---

### `main.go`
Entry point — loads config, opens DB, runs migrations, init queries, creates LLM client, initializes agent/tool registries, creates history repo + event bus + agent engine, builds handler, registers routes, logs endpoints, starts HTTP server with graceful shutdown.

#### Algorithm
Load config from `gopengai.json` (or `os.Args[1]` override). Open SQLite DB at `{DataDir}/gopengai.db` with WAL mode, FK enabled. Run Goose migrations. Init sqlc queries. Create LLM client from config. Initialize agent registry from `agents/` directory. Initialize tool registry with 4 tools (WebFetch, MemorySave, MemoryRecall, Delegate). Create history repository, event bus, agent engine. Build `api.Handler` with LLM client, queries, DB, config, engine, eventBus. Register all routes via `api.RegisterRoutes`. Log endpoint URLs. Start HTTP server in goroutine. Block on SIGINT/SIGTERM → graceful shutdown: HTTP drain (10s timeout) → EventBus.Close → queries.Close → database.Close.

#### Data
- **Flow:** config path → load config → open DB → migrate → init queries → create LLM client → init agent registry → init tool registry (4 tools) → create history repo → create event bus → create engine → build handler → register routes → start goroutine → wait for signal → graceful shutdown
- **Config:** `config.ServerConfig{Host, Port}`, `config.LLMConfig{Provider, BaseURL, APIKey, Model, MaxIterations}`
- **DB:** SQLite via `ncruces/go-sqlite3`, pooled connections, WAL mode
- **Routes:** `/session` (GET/POST), `/session/{id}` (GET/DELETE), `/session/{id}/message` (POST), `/v1/chat/completions` (POST), `/health` (GET)

#### Imports
- `gopengai/internal/api` — route registration, handler struct, EventBus
- `gopengai/internal/agent` — Engine, Registry, NewEngine
- `gopengai/internal/config` — `Load()`, config structs
- `gopengai/internal/db` — `Open()`, `Migrate()`, `New()`, queries
- `gopengai/internal/history` — `NewRepository()`
- `gopengai/internal/llm` — `NewClientFromConfig()`, client struct
- `gopengai/internal/tools` — `NewRegistry()`, WebFetchTool, MemorySave, MemoryRecall, DelegateTool
