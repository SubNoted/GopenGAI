API server entry point.

### `main.go`
Entry point — loads config, opens DB, runs migrations, init queries, creates LLM client, initializes agent/tool registries, creates history repo + event bus + agent engine, builds handler, registers routes, starts HTTP server with full graceful shutdown.

#### Algorithm
Parse CLI flags: `--config` (default `gopengai.json`) and `--port` (default 0, uses config value). Load config from file. Open SQLite DB at `{DataDir}/gopengai.db` with WAL mode. Run Goose migrations. Init sqlc queries. Create LLM client. Init agent registry from `agents/` dir (warn on fail). Init tool registry with 4 tools: WebFetch, MemorySave, MemoryRecall, DelegateTool. Create history repository. Create event bus. Create agent engine. Build `api.Handler` with LLM client, queries, DB, config, engine, eventBus, history. Register routes via `api.RegisterRoutes`. Create `http.Server` with `ReadHeaderTimeout: 10s`, `ReadTimeout: 30s`, `IdleTimeout: 120s` (no WriteTimeout for SSE). Create `quit` chan (buffer=1) for SIGINT/SIGTERM, `serverErr` chan (buffer=1) for listen errors. Start server in goroutine: log endpoints, call `ListenAndServe()`, send error to `serverErr` channel on failure (no `os.Exit`). Block on `select` with `quit` or `serverErr` → graceful shutdown: create 10s timeout ctx → `srv.Shutdown()` → `wg.Wait()` (waits for active engine goroutines) → `eventBus.Close()` → `database.Close()`.

#### Data
- **Flow:** Parse flags (`--config`, `--port`) → load config → open DB → migrate → init queries → create LLM client → init agent registry (warn on fail) → init tool registry (4 tools) → create history repo → create event bus → create engine → build handler → register routes → start goroutine → wait for signal/error → graceful shutdown (10s timeout, ordered: HTTP → wait wg → eventBus → DB)
- **Config:** `config.ServerConfig{Host, Port}`, `config.LLMConfig{Provider, BaseURL, APIKey, Model, MaxIterations}`
- **DB:** SQLite via `ncruces/go-sqlite3`, pooled connections, WAL mode
- **Handler fields:** `LLM`, `DB`, `SQLDB`, `Config`, `Engine`, `EventBus`, `Wg`, `History`
- **Routes:** `/session` (GET/POST), `/session/{id}` (GET/DELETE/PATCH), `/session/{id}/message` (POST), `/session/{id}/fork` (POST), `/session/{id}/branch` (PUT), `/session/{id}/abort` (POST), `/session/{id}/messages` (GET), `/session/{id}/branches` (GET), `/messages/{id}` (PATCH), `/events` (GET), `/session/{id}/events` (GET), `/agents` (GET), `/agents/{name}` (GET), `/memory` (GET), `/memory/{key}` (GET), `/v1/models` (GET), `/v1/chat/completions` (POST), `/health` (GET)
- **Shutdown seq:** `SetKeepAlivesEnabled(false)` → `srv.Shutdown(10s)` → `wg.Wait()` → `eventBus.Close()` → `database.Close()`
- **Middleware chain:** AuthMiddleware (bearer-token) → RecoveryMiddleware → CORSMiddleware → LoggingMiddleware
