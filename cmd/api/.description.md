### `cmd/api`

API server entry point.

---

### `main.go`
Entry point ‚Äî loads config, opens DB, runs migrations, init queries, creates LLM client, initializes agent/tool registries, creates history repo + event bus + agent engine, builds handler, registers routes, starts HTTP server with full graceful shutdown.

#### Algorithm
Load config from `gopengai.json` (or `os.Args[1]` override). Open SQLite DB at `{DataDir}/gopengai.db` with WAL mode. Run Goose migrations. Init sqlc queries. Create LLM client. Init agent registry from `agents/` dir (warn on fail). Init tool registry with 4 tools: WebFetch, MemorySave, MemoryRecall, DelegateTool. Create history repository. Create event bus. Create agent engine. Build `api.Handler` with LLM client, queries, DB, config, engine, eventBus. Register routes via `api.RegisterRoutes`. Create `http.Server`. Create `quit` chan (buffer=2) for SIGINT/SIGTERM. Start server in goroutine: log endpoints, call `ListenAndServe()`, use `log.Printf` + `os.Exit(1)` on error (deferred cleanup runs). Block on first signal ‚Üí graceful shutdown: create 10s timeout ctx ‚Üí `srv.Shutdown()` ‚Üí `eventBus.Close()` ‚Üí `queries.Close()` ‚Üí `database.Close()`. Second-signal goroutine: listen on `quit`, force `os.Exit(1)` if second signal arrives during shutdown.

#### Data
- **Flow:** config path ‚Üí load config ‚Üí open DB ‚Üí migrate ‚Üí init queries ‚Üí create LLM client ‚Üí init agent registry (warn on fail) ‚Üí init tool registry (4 tools) ‚Üí create history repo ‚Üí create event bus ‚Üí create engine ‚Üí build handler ‚Üí register routes ‚Üí start goroutine ‚Üí wait for signal ‚Üí graceful shutdown (10s timeout, ordered: HTTP ‚Üí eventBus ‚Üí queries ‚Üí DB)
- **Config:** `config.ServerConfig{Host, Port}`, `config.LLMConfig{Provider, BaseURL, APIKey, Model, MaxIterations}`
- **DB:** SQLite via `ncruces/go-sqlite3`, pooled connections, WAL mode
- **Handler fields:** `LLM`, `DB`, `SQLDB`, `Config`, `Engine`, `EventBus`
- **Routes:** `/session` (GET/POST), `/session/{id}` (GET/DELETE), `/session/{id}/message` (POST), `/v1/chat/completions` (POST), `/health` (GET)
- **Shutdown seq:** srv.Shutdown(10s) ‚Üí eventBus.Close() ‚Üí queries.Close() ‚Üí database.Close()
