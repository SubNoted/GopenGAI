### `cmd/api`
Directory containing the main entry point for the GoPengAI API server.

---

### `main.go`
Entry point for the API server application. Loads configuration from `gopengai.json` (with `os.Args[1]` fallback), opens SQLite database via `db.Open(path)` at `{DataDir}/gopengai.db`, runs Goose migrations via `db.Migrate(database)`, initializes sqlc-generated queries via `db.New(database)`, creates LLM client from config, creates API handler with the LLM client, queries, and config, registers routes via `api.RegisterRoutes(mux, handler)`, logs all endpoint URLs at startup, and starts HTTP server at configured host:port.

#### Exports
`main()` - Entry point that initializes the API server and starts listening on the configured host:port

#### Imports
- `gopengai/internal/api`
- `gopengai/internal/config`
- `gopengai/internal/db`
- `gopengai/internal/llm`

#### Endpoints Logged at Startup
- `POST /session` create session
- `GET /session` list sessions
- `GET /session/{id}` get session + messages
- `DELETE /session/{id}` delete session
- `POST /session/{id}/message` send message (history-aware)
- `POST /v1/chat/completions` OpenAI-compatible (no history)
- `GET /health` health check
