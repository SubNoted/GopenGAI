### `cmd/api`
Directory containing the main entry point for the GoPengAI API server.

---

### `main.go`
Entry point for the API server application. Loads configuration from `gopengai.json` (with `os.Args[1]` fallback), opens SQLite database via `db.Open(path)` at `{DataDir}/gopengai.db`, runs Goose migrations via `db.Migrate(database)`, creates LLM client from config, creates API handler with the LLM client, registers routes via `api.RegisterRoutes(mux, handler)`, and starts HTTP server at configured host:port.

#### Exports
| Export | Description |
|--------|-------------|
| `main()` | Entry point that initializes the API server and starts listening on the configured host:port |

#### Imports
- `gopengai/internal/api`
- `gopengai/internal/config`
- `gopengai/internal/db`
- `gopengai/internal/llm`
